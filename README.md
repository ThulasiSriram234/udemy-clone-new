
# 🎓 Udemy Clone Website

This project is a **Udemy Clone landing page** created using **HTML and CSS**.
It replicates the basic layout of the Udemy website including the navigation bar, course sections, and responsive design.

---

## 🚀 Features

* 📚 **Course Listing Section** – Displays available courses.
* 🧭 **Navigation Bar** – Includes logo, search bar, and menu links.
* 🎨 **Modern Layout** – Styled using CSS.
* 📱 **Responsive Design** – Works on desktop and mobile devices.
* 🖼️ **Course Cards** – Shows course image, title, and details.

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
